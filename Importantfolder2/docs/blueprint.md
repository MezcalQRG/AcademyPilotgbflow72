# **App Name**: LeadFlow Landing

## Core Features:

- Business Information Input: A form for users to input business details, including description, phone number, social media links, products, operation hours, and images.
- Genkit Flow Integration: Integrate a Genkit flow to adapt the inputted business information into a landing page format optimized for lead capture.
- Landing Page Templates: Display different landing page templates for users to preview how their information will be presented. Offer options for different design styles and layouts.
- Customizable Subdomain: Allows users to pick a custom subdomain based on their business name (e.g., businessname.leadflow.com) to host their landing page.
- Dynamic Content Injection: Inject the business information into the selected landing page template, creating a unique landing page for each user.
- Template Switching: Provide an option for users to switch to a different landing page template if they are not satisfied with the initial design.
- Page Visibility Toggle: A feature to make the page public, or hide the page.
